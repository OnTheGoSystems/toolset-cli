# wp views view add_loop_field

Adds a field to the end of View loop HTML.

### OPTIONS

&lt;id&gt;
: The post ID for View.

[\--field=&lt;string&gt;]
: Field shortcode


### EXAMPLES

   wp views view add_loop_field 42 --field="[post-excerpt]"
   wp views wpa add_loop_field 42 --field="[post-excerpt]"
